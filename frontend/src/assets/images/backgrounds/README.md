# backgrounds/

Imágenes de fondo para las distintas pantallas de la aplicación.

Archivos esperados:
- login-bg.jpg (o .png, .webp) — Fondo de la pantalla de login
- dashboard-bg.jpg — Fondo opcional del dashboard

Recomendaciones:
- Formato: JPG para fotos, PNG para imágenes con transparencia, WebP para mejor compresión
- Tamaño máximo recomendado: 500KB por imagen (optimizar antes de subir)
- Resolución mínima: 1920x1080px para fondos de pantalla completa

Uso en CSS:
  background-image: url('/assets/images/backgrounds/login-bg.jpg');

Uso en Angular template:
  <img src="assets/images/backgrounds/login-bg.jpg" alt="...">
